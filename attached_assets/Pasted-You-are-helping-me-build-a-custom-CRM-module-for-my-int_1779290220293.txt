You are helping me build a custom CRM module for my internal operations system.

PROJECT GOAL:
Build a fully custom HEAR / HER rebate program workflow module inside my CRM. This module should help my team manage every Neighborly rebate application from start to finish.

The module should live under the sidebar section:

Operations > Rebate Programs

The purpose is to replace our current spreadsheet with a structured case-management system.

--------------------------------------------------
MAIN FEATURE: REBATE PROGRAM CASE BANK
--------------------------------------------------

Create a main dashboard/table called “Rebate Program Cases.”

Each row represents one HEAR or HER application.

The table should include these columns:

1. Case Number
2. Client Name
3. Property Address
4. Program Type
   - HEAR
   - HER
5. Application Status
6. Current Workflow Step
7. Course of Action / Next Step
8. Assigned To
9. Priority
10. Last Updated
11. Expected Completion Date
12. Notes

Add a “Create New Case” button.

When creating a case, the user should select:

- Program Type: HEAR or HER
- Client Name
- Property Address
- Phone Number
- Email Address
- Case Number, if already known
- Assigned User

STATUS OPTIONS
--------------------------------------------------

Application Status should be a dropdown with these options:

- Not Started

- Scope of Work Submitted
- Returned for More Information
- Approved for Reservation
- Permit Needed
- Install Scheduled
- Project Complete
- Completion Submitted
- Rebate Reservation Summary Received
- Paid / Closed
- Declined
- Client Not Interested

--------------------------------------------------
COLOR CODING
--------------------------------------------------

Color code each case row based on status:

- Gray = Not Started
- Blue = In Progress
- Orange = Waiting on Client, Waiting on Internal Info, Returned for More Information, Approved for Reservation but still needs action
- Red = Urgent, Missing Required Info, or critical action needed
- Purple = Scope of Work Submitted
- Green = Approved, Project Complete, Paid / Closed
- Black = Declined or Client Not Interested
